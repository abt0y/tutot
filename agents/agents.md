- **Goal**: Build a fully client-side AI tutor using Rust/WASM.
- **Plan**: Follow `@implementation_plan.md`.

## Coding Standards
- Use `wasm-bindgen` for all browser-facing interfaces.
- Maintain strict modularity between the `agents` and `pedagogy` crates.
- Favor async Rust for all GitHub API interactions.
